# Support

Thanks for using **llm-armory**.

## Where to go

- **Questions & ideas** → [GitHub Discussions](https://github.com/muslewski/llm-armory/discussions)
- **Bugs & regressions** → [GitHub Issues](https://github.com/muslewski/llm-armory/issues/new/choose)
- **Security** → see [SECURITY.md](./SECURITY.md) (private report only)
- **Docs & install** → [README.md](./README.md) · website linked from the README

## Before you open an issue

1. Check existing issues and discussions for duplicates.
2. Confirm you're on the latest release (`armory --version` / package version).
3. Include OS, version, and the exact command that failed.
